# 框架的使用

This project is initialized with [Ant Design Pro](https://pro.ant.design). Follow is the quick guide for how to use.

## 环境准备

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

## Scripts配置


查看`package.json`. 

### 启动（本地）

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 代码检查

```bash
npm run lint
```

使用脚本自动修复一些lint 问题

```bash
npm run lint:fix
```

### 执行测试

```bash
npm test
```
